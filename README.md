## Expo?
>Expo算是React Native中的框架，能使開發人員不需接觸原生的程式碼，只用JS撰寫App，打比方來說，如果React Native是河中的踏腳石，Expo就是連接兩端的橋樑。
> 
> 使用Expo的優點
> 1. 只需撰寫JS，學習成本較低
> 2. 不需要有Android Studio或Xcode即可運行撰寫出的android或ios的app
> 3. 續2，亦即代表不用買macbook
> 
> React Native官網運行的demo也是使用Expo
